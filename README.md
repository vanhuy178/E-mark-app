### Ecommerce website using redux, router, bootstrap(you should use scss intead of bootstrap)

# Công nghệ sử dụng
    npm i react-router-dom
    npm i redux react-redux
    npm i font-awesome
    npm i boostrap - tiết kiệm thời gian(khuyến khít ae tự design - project này mình làm nhanh nên không có thời gian viết scss)
# Cấu trúc file
- src
    - components
        - Home.js
        - Navbar.js
        - Product.js
        - Loading.js
        - Product.js
        - ProductDetail.js
    - HOC
        - withRouter.js
    - redux
        - actions
            - cartTypes.js
            - index.js
            - .md
        - reducer
            - cartReducer.js
            - rootReducer.js
        - store.js
    - styles
        - loading.css
        - navbar.css
